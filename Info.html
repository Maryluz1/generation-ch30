A continuación te explico algunos conceptos de Java Script

·función
Una función es un bloque de código reutilizable que realiza una tarea específica. En programación, las funciones permiten dividir el código en fragmentos más pequeños y manejables, lo que facilita su mantenimiento y mejora la legibilidad. Las funciones pueden recibir parámetros (valores de entrada) y devolver un resultado (valor de salida).
·Let
`let` es una palabra reservada en JavaScript que se utiliza para declarar variables dentro del ámbito de un bloque de código. Las variables declaradas con `let` tienen un alcance local, lo que significa que solo son accesibles dentro del bloque en el que se declaran o en bloques anidados. Además, las variables declaradas con `let` pueden cambiar su valor durante la ejecución del programa.
·Const:
`const` es otra palabra reservada en JavaScript que se utiliza para declarar variables, pero a diferencia de `let`, las variables declaradas con `const` son constantes, es decir, su valor no puede cambiar una vez que se les asigna. Las variables `const` deben recibir un valor al ser declaradas y no pueden ser reasignadas posteriormente. Esto las hace útiles para definir valores que no deben cambiar durante la ejecución del programa.

Ahora te explicaré estas 3 etiquetas de Html

· Article:
Este Elemento de HTML representa una composición autocontenida en un documento, una página, una aplicación o en un sitio, que se quiere que sea distribuíble y/o reutilizable de manera independiente, por ejemplo, en la redifusión. Algunos ejemplos podrían ser un mensaje en un foro, un artículo de una revista o un periódico, una entrada de blog, el comentario de un usuario, un widget o gadget interactivo, o cualquier otro elemento de contenido independiente.
·Progress:
Esta etiqueta HTML se utiliza para visualizar el progreso de una tarea. Aunque los detalles de como se muestran dependen directamente del navegador que utiliza el cliente, aunque básicamente aparece una barra de progreso.
·Anside:
Este elemento HTML representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. Estas secciones son a menudo representadas como barras laterales o como inserciones y contienen una explicación al margen como una definición de glosario, elementos relacionados indirectamente, como publicidad, la biografía del autor, o en aplicaciones web, la información de perfil o enlaces a blogs relacionados .
